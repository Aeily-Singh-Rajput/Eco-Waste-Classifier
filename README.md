# ♻️ Eco-Waste Classifier
An AI-powered web application that classifies waste into different categories 
to promote eco-friendly disposal and recycling.

## 🚨 Problem Statement

Improper waste segregation leads to environmental pollution and inefficient recycling. 
Many people are unaware of how to correctly classify waste items.

## 💡 Solution

Eco-Waste Classifier uses machine learning to identify and classify waste 
from images into categories like recyclable, organic, and hazardous waste, 
helping users make better disposal decisions.

## ✨ Features

- 📷 Upload image of waste
- 🤖 AI-based classification
- ♻️ Suggests proper disposal method
- 🌐 User-friendly interface
